# REAL_AGENT Instructions

This folder contains a simple Python AI-style CLI agent that accepts any topic and returns **exactly 5 valid points**.

## Files
- `agent.py` → main runnable Python script.

## Requirements
- Python 3.8 or newer.

## How to Run
1. Open terminal in the project root (`/workspace/dev-core`).
2. Run:
   ```bash
   python3 REAL_AGENT/agent.py
   ```
3. Enter any topic when prompted.
4. Type `exit` (or `quit`) to stop.

## Example
Input topic:
```text
Artificial Intelligence in Healthcare
```

Output style:
```text
1. Definition: ...
2. Key Components: ...
3. Practical Importance: ...
4. Common Challenges: ...
5. Actionable Next Step: ...
```

## Notes
- The script always returns 5 points.
- It works offline and does not require API keys.
